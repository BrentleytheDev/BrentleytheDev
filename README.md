## Hi there, I'm Brentley 👋

### My story

I started learning computer science when I was 9 years old. I only learned basic HTML because my laptop kept crashing, so I waited until I got a PC before I started building projects. The languages I studied, in order, were HTML, Python, C++, C, and Java. I'm still early in my computer science journey and continue learning every day.

### My vision for software

I plan for privacy-focused open-source software and fast/efficient software.

### My dream

My dream is to attend MIT and build privacy-focused software that helps create a world with less unnecessary surveillance.
